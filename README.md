# Every Side Movers Website

Premium marketing website for Every Side Movers built with Next.js 14 App Router and Tailwind CSS.

## Tech Stack
- Next.js 14
- React 18
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Available Pages
- `/` Home
- `/services`
- `/pricing`
- `/about`
- `/contact`

## Deploy to Vercel
This project is deployment-ready for Vercel.
